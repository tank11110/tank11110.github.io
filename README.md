# Personal-Site

Personal portfolio site built with [Next.js 14](https://nextjs.org/) App Router and TypeScript.  
Bilingual (zh/en) with two pages: a collaboration-focused home and a printable resume.

## Quick Start

### Install dependencies

```bash
npm install
```

### Run the dev server

```bash
npm run dev
```

Site is running at → `http://localhost:3000`

<br>

## Project Structure

```
src/
├── app/           # Pages and root layout
├── components/    # Navbar and page sections
├── data/          # Personal content (edit here)
├── hooks/         # Language context and hook
└── types/         # Shared TypeScript types
```
